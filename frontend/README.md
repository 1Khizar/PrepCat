# PrepBuddy – Pakistan's Premier MDCAT & NUMS Platform

PrepBuddy is a state-of-the-art EdTech engine purpose-built for Pakistani medical aspirants. It combines precision study tools with AI-powered insights to ensure student success.

## 🚀 Phase 1 Foundation Completed
The platform's core infrastructure and visual identity have been established.

### Key Features
- **Stunning Landing Page**: Fully responsive with MDCAT countdown and interactive subject previews.
- **Premium Design System**: Deep Navy (#0D1B2A), Electric Teal (#00B4D8), and Sunset Gold (#FFB703).
- **Gamified Dashboard**: Personal streaks, study heatmaps, and progress rings.
- **Advanced Quiz Engine**: 4-option MCQs with instant feedback and AI-generated explanations via Groq LLM.
- **Onboarding Flow**: Multi-step registration and personalized study goal setting.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15+, TypeScript, Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Typography**: Inter & Plus Jakarta Sans

## 📂 Project Structure
- `/src/app`: App Router pages (Home, Login, Register, Profile, Quiz)
- `/src/components`: UI components (Navbar, Hero, Stats, etc.)
- `/src/app/globals.css`: Design tokens and global styles

## 📈 Development Roadmap
- **Phase 1**: Foundation & Core (Current)
- **Phase 2**: AI (Groq) Integration & Analytics
- **Phase 3**: Monetization & Social Features
- **Phase 4**: PWA & Offline Scale
